## 🚀 Quick Start

1. **Download Installation Package**: Download the latest `ok-of-win32-setup.exe` installation file from [GitHub Release](https://github.com/MarchSnow-1/ok-overfield/releases).
2. **Installer Program**: Double-click the `ok-of-win32-setup.exe` file and follow the installation wizard to complete the installation.
3. **Run Program**: After installation, launch `ok-of` from the desktop shortcut or the Start Menu.

## 🔧 Troubleshooting

If you encounter any problems, please check the following steps one by one before asking questions:

1. **Installation Path**: Ensure the software is installed in a **path with only English characters** (e.g., `D:\Games\ok-of`), and not in `C:\Program Files` or a folder containing Chinese characters
2. **Antivirus Software**: Add the software's installation directory to your antivirus software's **trusted zone or whitelist** to prevent accidental deletion or blocking of files
3. **Display Settings**:
    * Turn off all graphics card filters (such as NVIDIA Game Filter) and sharpening features
    * Use the game's default brightness settings
    * Turn off any overlays that display information on the game screen (such as frame rate displays in MSI Afterburner, Fraps, etc.)
4. **Software Version**: Check and ensure you are using the latest version of `ok-of`
5. **Game Performance**: Ensure the game runs stably at **60 FPS**. If the frame rate is unstable, try lowering the game's graphics settings or resolution
6. **Seek Help**: If the above steps do not resolve your issue, please submit a detailed bug report through community channels
