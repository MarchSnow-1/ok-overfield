## 🚀 快速开始

1.  **下载安装包**：[GitHub Release](https://github.com/MarchSnow-1/ok-overfield/releases)下载最新的 `ok-of-win32-setup.exe` 安装文件。
2.  **安装程序**：双击 `ok-of-win32-setup.exe` 文件，并按照安装向导的提示完成安装。
3.  **运行程序**：安装完成后，从桌面快捷方式或开始菜单启动 `ok-of` 即可。

## 🔧 疑难解答 (Troubleshooting)

如果遇到问题，请在提问前按以下步骤逐一排查：

1.  **安装路径**：请确保软件安装在**纯英文路径**下（例如 `D:\Games\ok-of`），不要安装在 `C:\Program Files` 或包含中文字符的文件夹中。
2.  **杀毒软件**：将软件的安装目录添加到您的杀毒软件（包括 Windows Defender）的**信任区或白名单**中，以防文件被误删或拦截。
3.  **显示设置**：
    *   关闭所有显卡滤镜（如 NVIDIA Game Filter）和锐化功能。
    *   使用游戏默认的亮度设置。
    *   关闭任何在游戏画面上显示信息的叠加层（如 MSI Afterburner、Fraps 等显示的帧率）。
4.  **软件版本**：检查并确保您使用的是最新版本的 `ok-of`。
5.  **游戏性能**：请确保游戏能稳定在 **60 FPS** 运行。如果帧率不稳定，请尝试降低游戏画质或分辨率。
6.  **寻求帮助**：如果以上步骤都无法解决您的问题，请通过社区渠道提交详细的错误报告。
