## 🚀 快速開始

1. **下載安裝包**：[GitHub Release](https://github.com/MarchSnow-1/ok-overfield/releases)下載最新的 `ok-of-win32-setup.exe` 安裝檔案。
2. **安裝程式**：雙擊 `ok-of-win32-setup.exe` 檔案，並按照安裝精靈的提示完成安裝。
3. **執行程式**：安裝完成後，從桌面捷徑或開始選單啟動 `ok-of` 即可。

## 🔧 疑難排解 (Troubleshooting)

如果遇到問題，請在提問前按以下步驟逐一檢查：

1. **安裝路徑**：請確保軟體安裝在**純英文路徑**下（例如 `D:\Games\ok-of`），不要安裝在 `C:\Program Files` 或包含中文字元的資料夾中。
2. **防毒軟體**：將軟體的安裝目錄新增到您的防毒軟體（包括 Windows Defender）的**信任區或白名單**中，以防檔案被誤刪或攔截。
3. **顯示設定**：

   * 關閉所有顯示卡濾鏡（如 NVIDIA Game Filter）和銳化功能。
   * 使用遊戲預設的亮度設定。
   * 關閉任何在遊戲畫面上顯示資訊的覆蓋層（如 MSI Afterburner、Fraps 等顯示的幀率）。
4. **軟體版本**：檢查並確保您使用的是最新版本的 `ok-of`。
5. **遊戲效能**：請確保遊戲能穩定在 **60 FPS** 執行。如果幀率不穩定，請嘗試降低遊戲畫質或解析度。
6. **尋求協助**：如果以上步驟都無法解決您的問題，請透過社群管道提交詳細的錯誤回報。
